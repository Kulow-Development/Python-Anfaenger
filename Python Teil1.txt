# Kommentare Hinzufügen
# Um Kommentare hinzuzufügen die keinen einfluss auf den Code haben benötigt man eine Raute wichtig ist aber dass Kommentare nicht in Semicolons gesetzt werden dürfen da diese sonst auch ausgegeben werden.
#Bsp:
print ("Willkommen in der Kulow Famile! #Begrüßung") # So wäre dass Falsch
print ("Willkommen in der Kulow Familie!") # Begrüßung # So wäre dass Richtig

# Mit dem folgenden Befehl lassen sich Texte zahlen ausgeben  
print ("text") # Dieser Befehl wird dass Wort "text" ausgeben

# Zudem sollte man beachten dass es 2 Semicolons gibt einmal " und ' diese dienen um Fehler zu vermeiden
